IP Geomaster Documentation

Introduction

IP Geomaster is a lightweight yet powerful WordPress plugin designed to block access to your website from specific countries, IP addresses, and malicious bot traffic. With this plugin, you can effortlessly manage which countries, IP addresses, and bots are allowed or banned from accessing your site, providing you with full control over your website's security, performance, and overall accessibility.

By blocking unwanted traffic, you can significantly reduce the risk of attacks, ensure faster loading speeds, and monitor threats in real-time. This enhances not only your website's security but also its performance and user experience.

Installation

Follow these steps to install and activate IP Geomaster:

  1. Upload the ip-geomaster.php file to the /wp-content/plugins/ directory.
  2. Activate the plugin through the 'Plugins' menu in WordPress.
  3. Once activated, you can access the plugin settings in the WordPress admin dashboard under the "IP Geomaster" menu.

Plugin Features

  • Block or allow access based on country, IP address, or bot traffic.
  • Manage blocked IPs directly from the plugin dashboard.
  • Real-time threat monitoring and reporting.
  • Customizable settings to suit your website's needs.
  • Fast and lightweight with minimal impact on website performance.

How to Use

After installing and activating the plugin, follow these steps to block or allow IP addresses:

  1. Go to the plugin settings page under the "IP Geomaster" menu in your WordPress admin.
  2. Select the action you want to perform (block or allow) for countries, IP addresses, or bot traffic.
  3. To block specific IP addresses, simply add them to the list of blocked IPs.
  4. You can also view the list of all currently blocked IPs and remove them as needed.

Frequently Asked Questions

How can I unblock an IP address?

To unblock an IP, go to the "Blocked IPs" section in the plugin settings and remove the IP from the list. The IP will immediately regain access to your website.

Can I block entire countries?

Yes, IP Geomaster allows you to block or allow traffic from specific countries using GeoIP data.
